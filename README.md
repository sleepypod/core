# sleepypod

## Getting Started

### Prerequisites

- Node.js LTS (16.9+)
- Corepack (included with Node.js)

### Installation

```bash
# Enable Corepack for Yarn
corepack enable

# Install dependencies (uses offline mirror - fast!)
yarn install
```

### Development

```bash
yarn dev          # Start development server
yarn build        # Build for production
yarn test         # Run tests
yarn lint         # Lint code
yarn tsc          # Type check
```
