# ADR: Vitest for Testing

## Context
We chose Vitest as the testing framework for this project due to its modern features and compatibility with Next.js.

### 1. Next.js Support
- Vitest provides excellent support for Next.js, making it a natural choice for our stack.
- It integrates well with the React ecosystem, ensuring smooth testing workflows.

### 2. Future Support for Storybook
- Vitest's compatibility with Storybook ensures that we can easily add visual testing in the future.
- This forward-looking choice aligns with our goal of maintaining a modern and flexible testing setup.
